01

Data yang kami proses

Data Diproses? Di mana
Foto & video di galeri Ya, on-device Hanya di ponsel Anda, untuk fitur swipe & hapus
Riwayat review (keep/delete) Ya, on-device Database lokal aplikasi
Crash logs & info perangkat Ya (Firebase Crashlytics) Google/Firebase — untuk stabilitas app

Kami tidak mengumpulkan foto, video, lokasi GPS/EXIF, atau kontak Anda.

02

Izin aplikasi

  • Akses foto/video: untuk menampilkan dan menghapus media yang Anda pilih via swipe
  • Internet: hanya untuk mengirim laporan crash anonim (Crashlytics)
03

Penghapusan data

  • Hapus foto: melalui alur review app → konfirmasi Android → Recently Deleted sistem (±30 hari)
  • Hapus data lokal app: Settings → Hapus data lokal
04

Anak-anak

DeleteIt tidak ditujukan untuk anak di bawah 13 tahun.

05

Perubahan kebijakan

Kebijakan ini dapat diperbarui. Versi terbaru akan dipublikasikan di halaman ini.

06

Kepatuhan regulasi

UU PDP (Indonesia): DeleteIt tidak memproses data pribadi ke server operator. Pemrosesan media sepenuhnya di perangkat pengguna.

GDPR: Data terbatas pada crash diagnostics (Firebase). Bukan konten foto/video.
